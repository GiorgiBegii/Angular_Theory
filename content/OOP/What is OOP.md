OOP (Object-Oriented Programming) is a programming style where we organize code around objects. An object contains data (properties) and behavior (methods) that belong together.

The goal of OOP is to make code easier to understand, reuse, maintain, and extend.

OOP is a way of organizing code using objects that combine data and behavior. Its main principles are encapsulation, inheritance, polymorphism, and abstraction, which help create reusable, maintainable, and scalable applications.

### Four main OOP principles

1. Encapsulation - hide internal details and expose only what is needed.
2. Inheritance - create new classes based on existing ones.
3. Polymorphism - use different implementations through the same interface.
4. Abstraction - focus on what an object does, not how it does it.
